Negrito e Italico

Negrito

Para usar o negrito dever usar a tag semantica (STRONG).

Italico

Para usar o italico deve usar a tag semantica (EM).

Texto marcado

Tambem e possivel usar um texto marcado usando a tag MARK.

texto deletado

E possivel fazer uma tag de texto deletado com a tag DEL.

Texto inserido

E possivel tambem fazer uma tag de texto sublinhado ou inserido usando a tag INS.

Texto sobrescrito

Para adicionar um texto sobre-escrito EX: x20+3 tem q usar a tag SUP.

Texto subscrito

Para inserir coisa do tipo H2O deve usar a tag SUB.
